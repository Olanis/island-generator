# Insel-Generator

Dieses Projekt ist der Anfang eines PC-Videospiels. Ziel ist es, einen Inselgenerator zu bauen, der prozedurale Inseln erstellt, auf denen ein Charakter laufen kann.

## Erster Schritt: Einfacher Insel-Generator

- Ein Button "Insel generieren"
- Generiert zufällig entweder ein Quadrat oder ein Rechteck als 3D-Objekt
- Visualisierung mit matplotlib (3D-Plot)

## Installation

1. Clone das Repo
2. Installiere Dependencies: `pip install -r requirements.txt`
3. Starte: `python main.py`
